# 개인정보처리방침 — 인프런 자막 다운로더

**최종 수정일:** 2026년 5월 3일

## 개요

인프런 자막 다운로더(이하 "본 확장 프로그램")는 인프런(inflearn.com) 강의의 스크립트 탭에 자막 다운로드 버튼을 추가하는 Chrome 확장 프로그램입니다. 본 확장 프로그램은 사용자의 어떠한 개인정보도 수집, 저장, 전송하지 않습니다.

## 수집하는 데이터

없음. 본 확장 프로그램은 개인 식별 정보, 사용 기록, 인증 정보 등 어떠한 사용자 데이터도 수집하지 않습니다.

## 권한 사용 목적

- **호스트 권한 (`https://www.inflearn.com/*`)**: 인프런 강의 페이지에 자막 다운로드 버튼을 삽입하기 위한 콘텐츠 스크립트 실행 용도로만 사용됩니다. 인프런 외의 다른 웹사이트에는 접근하지 않습니다.

## 데이터 처리 방식

- 자막 데이터는 사용자가 인프런에서 강의를 재생할 때 브라우저가 인프런 서버로부터 정상적으로 수신하는 API 응답을 로컬에서 캡처하여 추출합니다.
- 추출된 자막은 사용자가 다운로드 버튼을 눌렀을 때 파일로 저장되며, 그 외 어떠한 외부 서버로도 전송되지 않습니다.
- 캡처된 데이터는 브라우저 메모리에만 일시적으로 존재하며, 탭 또는 브라우저 세션 종료 시 즉시 소멸됩니다.
- `chrome.storage`를 포함한 어떠한 영구 저장소도 사용하지 않습니다.

## 외부 서비스

본 확장 프로그램은 자체 서버, 외부 API, 서드파티 분석 도구, 광고 네트워크 등 어떠한 외부 서비스와도 통신하지 않습니다.

## 정책 변경

본 개인정보처리방침이 변경될 경우 본 문서의 최종 수정일과 함께 업데이트됩니다.
